//==================================
  Алгоритм розгортання додатку на локальному сервері з безпосереднім доступом
1. Налаштування веб сервера.
2. Розгорнути базу.
3. Розгорнути та налаштувати додаток

//----------------------------------
Важливо!
 Перед початком розгортання додатку
 на сервері має бути встановлене і сконфігуроване
 програмне забезпечення вказане на початку. 
//----------------------------------

1. сконфігурувати веб сервер Apache параметер
   DirectoryIndex  index.php(додати параметр) index.html

2. Процес розгортання бази.

  1.1 Запустити та налаштувати phpmysqladmin, а саме:
     * назву хоста MySQL  або IP адресу
     * назву бази даних ( в нашому випадку chocoplanet)
     * імена та паролі на доступ до Mysql Server 
  
  1.3 Створити папку (буква диску):\базаданих\ і скопіювати в неї архів з базою даних             (диск://Enviropment/DataBase/batabase.zip), розпакувати архів.

  1.3 за допомогою візарда phpmysqladmin імпортувати файл бази даних,
      що знаходиться в архіві (диск://Enviropment/DataBase/batabase.zip)
      (при імпорті жодних налаштувань не слід змінювати).

  1.4 В процесі розгортанн не повнно виникнути жодних помилок, але
      у разі їх виникнення звертайтесь за підтримкою тел: 123-123-123

  1.5 Всі налаштування безпеки виконує адміністратор вашої системи.

      
3. Процес розгорння та налаштування додатоку.
  
  2.1 розпакувати дистрибутив додатку (знаходиться в папці        диск://Enviropment/Application/chоcoplаnet.zip ) в папку
      де будуть хоститись сайти на веб сервері Apache(htdocs, www, або іншу в залежностіві дваших налаштувань)
  
  2.2 в результаті розпаковки архіву має зявитися 2 папки (/system, /application) і файл index.php   
  
  2.3 налаштувати додаток для роботи з базою даних:
    а) зайти впапку application/config/ і відредагувати файл
       database.php
       в ньому змінити відмічені поля

   $db['default']['hostname'] = "localhost"; //прописати імя хоста або IP адресу
   $db['default']['username'] = "root";  // прописати імя користувача бази даних
   $db['default']['password'] = "";      // прописати пароль користувача бази даних
   $db['default']['database'] = "database_name"; // прописати імя бази даних
   $db['default']['dbdriver'] = "mysql";
   $db['default']['dbprefix'] = "";
   $db['default']['pconnect'] = TRUE;
   $db['default']['db_debug'] = FALSE;
   $db['default']['cache_on'] = FALSE;
   $db['default']['cachedir'] = "";
   $db['default']['char_set'] = "utf8";
   $db['default']['dbcollat'] = "utf8_general_ci";
   $db['default']['swap_pre'] = "";
   $db['default']['autoinit'] = TRUE; 

  2.4 а) зайти впапку application/config/ і відредагувати файл
       config.php
  
    config['base_url']	= 'www.sitename.net';  //прописати імя вашого сайту
