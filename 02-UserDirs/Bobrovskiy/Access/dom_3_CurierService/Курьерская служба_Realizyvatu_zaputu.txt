Вся система состоит из следующих форм:
1.	Курьеры
2.	Клиенты
3.	Рассылки по клиентам
4.	Наряды на доставку
5.	Расчеты с курьерами

1.	Форма «Курьеры» состоит из полей:

+1.1.	Фамилия
+1.2.	Имя
+1.3.	Отчество
+1.4.	Дата рождения
+1.5.	Паспортные данные
+1.6.	Адрес места жительства
+1.7.	Телефон домашний
+1.8.	Телефон мобильный
+1.9.	Возможное время работы по дням недели и часам
+1.10.	Примечания
1.11.	Итого расчеты с курьером (таб. 5)   (только отображение, вычисляемая величина)
+1.12.	Дата заключения договора
+1.13.	Дата расторжения договора

2. Форма «Клиенты» состоит из полей:

+2.1. Наименование клиента
+2.2. Контактное лицо
+2.3. Телефоны
+2.4. Адрес
+2.5. Адрес Эл. Почты

3. Форма «Рассылки по клиентам» состоит из полей:

3.1.	Наименование рассылки
3.2.	Клиент (таб. 2)
3.3.	Начало рассылки (дата)
3.4.	План. окончание рассылки (дата)
3.5.	Список нарядов на доставку по рассылке (таб. 4)
3.6.	Сумма затрат на дорогу по рассылке (сумм. 4.12 по всем нарядам, только отображ.)
3.7.	Статус рассылки (новая/текущая/закрытая)

4.	Форма «Наряды на доставку» состоит из полей:

4.1.	Номер наряда на доставку
4.2.	Дата выдачи путевого листа
4.3.	Курьер (таб. 1)
4.4.	Рассылка (таб. 3)
4.5.	Номер выданного путевого (3 цифры)
4.6.	Кол-во адресов на доставку по путевому
4.7.	Ставка оплаты за единицу отправления
4.8.	Статус наряда (новый/забронирован/на руках/ закрыт)
4.9.	Дата закрытия
4.10.	Кол-во адресов доставлено факт.
4.11.	Начислено к оплате, грн. (вычисл. Как 4.10*4.7, только отображение)
4.12.	Выдано за дорогу, грн.

5.	Форма «расчеты с курьерами» состоит из полей:

5.1.	Курьер (таб. 1)
5.2.	Начислено к выдаче (сумм. 4.11 по всем нарядам со статусом «закрыт», толко отображение)
5.3.	Дата заказа на выдачу з/п (вводится менеджером курьеров)
5.4.	Дата фактической выдачи з/п
5.5.	Выдано, руб.
5.6.	Итого по курьеру (5.2-5.4)


Запросы:
1. Курьеры с которыми не расторгнут договор
2. Курьеры с которыми расторгнут договор
3. Рассылки со статусом "Новая"
4. Наряды со статусом "Закрыт" и у которых с курьером выполнявшим рассылку расторгнут договор
5. Курьеры имена которых содержат в себе букву "в"
6. Клиенты, адрес эл. почты которых находится на mail.ru
7. Рассылку у которой максимальная сумма затрат на дорогу
8. Наряд у которого минимальное значение "Выдано на дорогу"
9. Рассылки вместе с клиентами, группированными по имени клиента в алфавитном порядке
10. Наряды на доставку вместе с курьерами у которых "Дата выдачи путевого листа" попадает в заданный диапазон
